# Vet App

### Uso

Instalar dependencias

```bash
npm install
```

Crear un archivo `.env.local`. Con las siguientes variables de entorno:

```bash
VITE_AUTH0_DOMAIN=
VITE_AUTH0_CLIENT_ID=
VITE_API_PORT=5555
```

Correr servidor:

```bash
npm run server
```

```bash
npm run dev
```
